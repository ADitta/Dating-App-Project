# Description

This is a full-stack application built using Angular SPA for the frontend and a .NET API for the backend. This is a feature rich application that contains the following features which allow users to:
<ul>
    <li>Register/Login</li>
    <li>Change the theme of the site</li>
    <li>View other users</li>
    <li>Like other users</li>
    <li>View users who liked them and mutual likes</li>
    <li>Upload photos to a cloudinary server</li>
    <li>View list of uploaded photos and set any approved photo as their main profile picture</li>
    <li>View their inbox/outbox of messages</li>
    <li>Recieve error/success/info toast notifications</li>
    <li>Communicate with other users with real time messaging using SignalR</li>
    <li>View whether other users are online</li>
    <li>Filter users by age, gender and order by last active/newest members</li>
    <li>Dynamically update the list of users with pagination</li>
    <li>Quickly navigate through previously loaded pages by caching responses from the Api </li>
    <li>If a moderator/admin, the user is allowed to approve photos (this has been disabled on the hosted app to allow any users to test the photo functionality easily) and grant other users moderator/admin status </li>
</ul>
